# Universal Game Score Tracker

A simple web app for tracking scores across multiple games and players.

## Features

- Create and manage multiple games
- Add players to each game
- Track individual player scores with history
- Set winning point targets
- Built-in calculator for score adjustments
- Persistent storage using browser localStorage

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Tech Stack

- React + TypeScript
- Vite
- CSS Modules
